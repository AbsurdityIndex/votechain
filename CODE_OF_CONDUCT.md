# Code of Conduct

VoteChain is committed to being a respectful, inclusive, and harassment-free project for everyone.

## Expected behavior

- Be respectful, kind, and constructive in all project spaces.
- Keep feedback focused on the code, the protocol, and the problem being solved.
- Critique ideas and proposals directly, without personal attacks.
- Respect different perspectives and be patient with newcomers.
- Report issues promptly if you witness harmful behavior.

## Unacceptable behavior

- Harassment, threats, hate speech, or discrimination.
- Doxxing, posting private information, or targeting contributors.
- Repeated trolling, intimidation, or deliberate disruption.
- Spamming or abusing issue and pull request discussions.

## Enforcement

Project maintainers may remove, block, or ban participants who repeatedly violate these expectations. Appeals can be sent to the maintainer team through the contact method in [SECURITY.md](SECURITY.md).

## Scope

This code applies to all project spaces, including:

- GitHub issues and pull requests
- Repository discussions
- Any public or private communication directly tied to project contributions
